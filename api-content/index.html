前五年的先拂是纯铁路存档，除铁路之外再无他物。铁路直接裸露铺在地上，无任何围挡；「车站」仅由停车装置和站牌构成，连站台也没有。虽然简陋如此，但五年来一共修了14条线路（1~14号线），250余座车站，基本覆盖了半径1500范围的广阔区域。
